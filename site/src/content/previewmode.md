# previewmode.cfg

> 饰品预览检视工具配置文件

- 在 `autoexec.cfg` 中按 `9` 键启用
- 功能：方便拍照截图检视的预设，需要在自建房/作弊服务器上使用

## 首次加载时的预设

- 关闭 HUD
- 开启景深
- 拉长 FOV 视角
- 固定摄像机镜头

## 功能表

| 功能 | 快捷键 | 控制台别名 |
| :---: | :---: | :---: |
| FOV 视角调节 | `↓` | - |
| 向左旋转视角 | `-` | - |
| 向右旋转视角 | `=` | - |
| 开关 HUD | `z` | - |
| 是否固定摄像机视角 | `x` | - |
| 是否开启景深 | `c` | - |
| 调节景深深度 | `v` | - |
| 取消预设值，恢复如初 | - | `depre` |
