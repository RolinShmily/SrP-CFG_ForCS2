Core Features

核心功能

覆盖从路径检测到备份恢复的完整安装链路，每一步都自动化处理

路径自动检测

自动扫描注册表与 Steam 库文件，精准定位游戏全局 CFG 路径和地图指南路径，无需手动翻找

用户自动识别

自动枚举 Steam 用户列表并识别当前登录用户，多账号环境支持手动切换选择

智能备份

安装前自动将全局 CFG、用户视频预设、地图指南分别打包为独立 ZIP 备份，随时可回滚

拖拽安装

将 ZIP 配置包、CFG 单文件或整个文件夹直接拖入窗口，自动识别类型并完成安装

多安装目标

全局 CFG、用户视频设置、地图指南三个独立目标，通过复选框自由组合安装范围

实时日志

颜色编码的实时日志输出，安装进度清晰可辨：绿色成功、橙色警告、默认信息

自动更新

启动时后台检查 GitHub Releases 新版本，有更新自动提醒，支持忽略当前版本

零依赖运行

自包含单文件发布，无需安装 .NET 运行时或任何其他依赖，下载即可直接运行

In Action

实际使用

安装器的完整工作流程，从路径检测到文件部署

路径检测

自动读取注册表获取 Steam 安装路径，解析 libraryfolders.vdf 定位 CS2 游戏目录，枚举 userdata 下的用户列表

备份机制

安装前自动将现有配置打包为 ZIP 备份文件，支持一键打开备份所在目录，可随时回滚到之前的状态

拖拽识别

自动判断输入类型：ZIP 包执行解压安装，单文件直接复制到位，文件夹递归部署所有内容

实时反馈

安装过程中实时显示操作日志，颜色编码区分操作状态：绿色成功、橙色异常、默认信息

独立控制

全局 CFG、视频设置、地图指南三个目标通过复选框独立控制，按需选择安装范围

更新通知

启动时后台检查新版本，有更新时显示通知横幅，可忽略当前版本或一键跳转下载页
