> 武器自适应视角切换配置文件

## 简介

autoview.cfg 实现武器切换时自动切换持枪视角。主武器使用独立的自定义视角预设（默认 v06），切换至副武器、匕首、电击枪、投掷物等其他装备时自动切换为默认 M7 视角（v00）。

此功能适合希望主武器有更开阔的视野、同时其他装备保持标准视角的玩家。

## 激活方式

在 autoexec.cfg 中按 `[` 键启用。

## 工作原理

加载后，按键 1-6 和 z/x/c/v 会被重绑，在切换武器的同时自动调用视角切换 alias：

- 主武器（按键 1）调用 `view_0`，默认为 v06
- 其他装备（按键 2-6, z, x, c, v）调用 `view_1`，默认为 v00

## 受影响的按键

| 按键 | 武器槽位 | 视角预设 |
| :--- | :--- | :--- |
| 1 | 主武器 | view_0 (v06) |
| 2 | 副武器 | view_1 (v00) |
| 3 | 匕首 | view_1 (v00) |
| 4 | 电击枪 | view_1 (v00) |
| 5 | C4 | view_1 (v00) |
| 6 | 诱饵弹 | view_1 (v00) |
| z | 高爆手雷 | view_1 (v00) |
| x | 闪光弹 | view_1 (v00) |
| c | 烟雾弹 | view_1 (v00) |
| v | 燃烧弹 | view_1 (v00) |

## 自定义

如需更改主武器视角预设，编辑 autoview.cfg 中的 `view_0` alias，将 `v06` 替换为 v00-v07 中的任意预设。同理可修改 `view_1` 来更改其他装备的视角。

## 相关文件

- [crosshair_view.cfg](/docs/crosshair_view) — 视角预设定义（v00-v07）
- [autoexec.cfg](/docs/autoexec) — 主配置，按 `[` 键加载本文件

## 注意事项

- 必须搭配 crosshair_view.cfg 使用，视角预设 v00-v07 由其定义
- 无法实现 Q 键切换装备时的视角更换（CS2 引擎限制）
- 加载后按键 1-6/z/x/c/v 的绑定会被覆盖，重启 autoexec.cfg（按 O 键）可恢复默认
