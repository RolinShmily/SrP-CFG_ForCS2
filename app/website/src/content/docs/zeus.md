> 电击枪快速切换配置文件

## 简介

zeus.cfg 实现电击枪使用后自动切回主武器或副武器。左键发射电击枪后自动切换回枪械，手持电击枪时右键则不使用电击枪、直接切回枪械。

该功能避免了电击枪使用后需要手动切枪的延迟，在实战中能更快恢复火力。

## 激活方式

在 autoexec.cfg 中默认不启用。有两种激活方式：

1. 在 autoexec.cfg 中取消 `exec zeus.cfg` 的注释
2. 在控制台输入 `zeus`

## 工作原理

通过重绑鼠标按键实现自动切换：

1. 按下 4 键选择电击枪时，mouse1/mouse2 被替换为自定义 alias
2. 左键释放后自动执行 `slot2;slot1` 切回枪械
3. 右键按下时直接执行 `slot2;slot1`（不触发电击枪攻击）
4. 切回后自动恢复 mouse1/mouse2 的默认攻击绑定

同时重绑按键 1-5 和 Q 键，确保从电击枪切换到任意武器后按键状态正确。

## 受影响的按键

| 按键 | 功能变化 |
| :--- | :--- |
| 1 | 主武器 + 恢复攻击绑定 |
| 2 | 副武器 + 恢复攻击绑定 |
| 3 | 匕首 + 恢复攻击绑定 |
| 4 | 选择电击枪 + 启用自动切换机制 |
| 5 | C4 + 恢复攻击绑定 |
| Q | 切换最近武器 + 恢复攻击绑定 |

## 相关文件

- [autoexec.cfg](/docs/autoexec) — 主配置，默认绑定 4 键为电击枪 (slot11)

## 注意事项

- 加载后按键 1-5 和 Q 的绑定会被覆盖，重启 autoexec.cfg（按 O 键）可恢复默认
- 4 键必须绑定为电击枪（slot11），否则自动切换不会触发
- 电击枪充电完成有声音提示，注意利用该声音做战术调整，或直接丢弃电击枪避免暴露位置
