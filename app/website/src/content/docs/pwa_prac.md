## 简介

完美世界对战平台的跑图服务器禁用了大部分客户端指令（如 `sv_cheats` 及其衍生命令），转而通过聊天栏指令（如 `.bot`、`.qy`、`.tz` 等）来控制跑图环境。

`pwa-prac`（Perfect World Practice）模式通过配置 `say` 别名并绑定按键，完美适配了完美平台的跑图服务器功能。本模式在 `srp_pwa_prac_keys` 时接管你的工作区键位，让你可以像在本地 `practice` 模式一样，一键管理 Bot 和投掷物，无需手动输入聊天命令。

## 激活方式

推荐在进入完美跑图服务器后，按下 `Home` 键，或者在控制台输入 `srp_pwa_prac_keys`。该命令会应用完美跑图的专用快捷键。

## 可选按键表

下表只有在执行 `srp_pwa_prac_keys` 后才生效。

| 功能 | 快捷键 | 完美世界指令 |
| :--- | :--- | :--- |
| 道具训练 | `F1` | `.u` |
| 上帝模式 | `F2` | `.god` |
| 追踪道具 | `F3` | `.cam` |
| 指令墙 | `F4` | `.menu` |
| 倒数第2颗道具复现 | `F5` | `.ct2` |
| 倒数第3颗道具复现 | `F6` | `.ct3` |
| 倒数第4颗道具复现 | `F7` | `.ct4` |
| 倒数第5颗道具复现 | `F8` | `.ct5` |
| 清除所有 Bot | `K` | `.kickall` |
| 友伤状态开关 | `F9` | `.ys` |
| 破坏物品 | `F10` | `.break` |
| 恢复物品 | `F11` | `.recover` |
| 飞行模式 | `N` | `noclip` |
| 最近1颗道具复现 | `L` | `.tz` |
| 清除烟雾弹+闪光效果 | `,` | `.qy` + `.sg` |
| 添加 Bot | `MOUSE4` | `.bot` |
| 清除指向的 Bot | `↓` | `.kick` |
| 预览窗+子弹落点 | `↑` | `.up` + `.bi` |
| 重置游戏 | `→` | `.cz` |
